Data Repository

Forage fish: This program is part of the Coastal Forage Fish Network. These data are deposited with the Pacific Salmon Foundation and can be accessed at https://marinedata.psf.ca/knowledge-hubs/forage-fish/

Salmon Escapement Monitoring: Data shared with Fisheries and Oceans Canada. These data appear in the periodic South Coast Salmon Strait of Georgia Stock Assessment.

Seagrass Monitoring: Data not currently shared.

Intertidal Macroinvertebrates and Seaweeds Monitoring: Data not currently shared.

Pinniped monitoring: Data not currently shared.

Coastal Waterbirds: Data shared with Bird Studies Canada's Nature Counts and eBird.

Loon Monitoring: Data shared with Bird Studies Canada Loon Lakes Monitoring Program.

Larval Crab Monitoring: Data shared with the Hakai Institute's Sentinels of Change program and the Pacific Northwest Crab Research Group.

Invasive Species Including European Green Crab and Clubbed Tunicates: Data shared with Fisheries and Oceans Canada.
