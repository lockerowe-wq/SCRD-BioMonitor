The Loon Foundation: Data Repository

  • Forage fish: This program is part of the Coastal Forage Fish Network (https://projectwatershed.ca/2024/02/12/coastal-forage-fish-network/). These data are deposited with the Pacific Salmon Foundation and can be accessed at https://marinedata.psf.ca/knowledge-hubs/forage-fish/
  • Salmon Escapement Monitoring: Data shared with Fisheries and Oceans Canada. Access not currently known.
  • Seagrass Monitoring: Data not currently shared.
  • Rocky Intertidal: Data not currently shared.
  • Pinniped monitoring: Data not currently shared.
  • Coastal Waterbirds: Data shared with Bird Studies Canada's Nature Counts (https://naturecounts.ca/nc/bccws/main.jsp?) and eBird.
  • Loon Monitoring: Bird Studies Canada Loon Lakes Monitoring Program https://www.birdscanada.org/bird-science/canadian-lakes-loon-survey
  • Larval Crab Monitoring: Data shared with the Hakai Institute's Sentinels of Change program (https://sentinels.hakai.org/approaches/light-traps) and the Pacific Northwest Crab Research Group (https://www.pnwcrab.com/larval-crab-monitoring)
  • Green Crab Monitoring: Data shared with Fisheries and Oceans Canada.
  • Anderson Creek Water Quality: Data not currently shared.
